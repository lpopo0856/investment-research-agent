本代码仓库是在你电脑本地运行的 **AI 投资研究助手**工作区。在 **Claude Code、OpenAI Codex、Gemini CLI**，或任何能读文件、跑终端命令的 Agent 环境中打开本仓库后，直接用自然语言对话即可。

**模型建议：** 使用 **Claude Sonnet 4.6**，推理强度设为 **High**，或改用同等或更强的更新模型。较轻的模型更容易跳过步骤或拉低分析深度。

## 直接跟助手说就行

不必先背命令、字段格式或目录结构。按你想做的事，复制下面任意一块发给助手即可。

**第一次用？**

> 「带我入门。」*（或附上券商对账单／交割明细——PDF、CSV、JSON、XLSX、截图、粘贴的文字均可——并说「帮我做 onboarding」）*

**想看看这里能做什么？**

> 「这里能做什么？」

**调整助手如何替你记账与表态（风险偏好、仓位大小、禁区、语言、本位币种）：**

> 「带我过一遍设置。」
> 「看下我的 SETTINGS。」／「把我的基准货币改成 TWD。」

**记一笔买卖或资金进出：**

> 「昨天 185 美元买了 30 股 NVDA。」
> 「今天 400 美元卖掉 10 股 TSLA。」
> 「GOOG 一季度分红 80 美元。」
> 「往账户里转了 5,000 美元。」／「入金 5,000 美元。」
> 「这是我从 Schwab 导出的 CSV——请导入。」（其他券商导出格式亦可，按 `docs/` 下约定处理）

**导入提示：** 持有台股时，若有台湾证券交易所（TWSE）导出文件，请优先附上。PDF 已加密时，请先用浏览器打开，再用浏览器的**打印**另存为无密码的 PDF，再交给助手导入。交易文件很大时（尤其是 PDF），请拆成较小的文件，分批导入。

**问研究／持仓相关问题：**

> 「结合我现在的持仓分析 NVDA。」
> 「我现在和 AI 相关的敞口有多大？」
> 「财报季前要不要先减掉短线仓位？」

**生成今天的组合报告：**

> 「生成今天的组合体检报告。」
> 「跑一版盘前报告。」

会改动你已保存数据的操作，都会先请你确认。用平常说话的方式提需求即可；助手会按 `docs/` 约定全流程执行并处理细节。

## 隐私

你的配置文件、交易数据库（SQLite）以及每一份生成的报告都留在本机，**不会**被 Git 追踪或上传。纳入版本控制的只有助手规格说明、示例模板与 Python 脚本。

## 第三方数据

行情流程可能调用公开市场数据与汇率接口（例如 Stooq、Yahoo、Binance、CoinGecko、Frankfurter／欧洲央行 ECB、Open ExchangeRate-API，以及 **台湾地区 TWSE／TPEx** 等），外加你自行配置的可选 API 密钥。本项目不运营也不替任何数据商背书——用户条款、调用频率与付费额度请自行核实并负责。
