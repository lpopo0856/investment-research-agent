本仓库是 AI 投资研究代理的本地工作区。在 **Claude Code、OpenAI Codex、Gemini CLI**，或任何能读文件并运行命令的代理环境中打开此 repo，直接和它对话即可。

**模型层级：** 请使用 **Claude Sonnet 4.6** 并开启 **High** 推理强度，或任何同等或更高能力的较新模型。较轻的模型可能会跳过步骤或削弱深度。

## 直接跟代理说

你不必先学命令、格式或文件结构。按你想做的事，复制下面任意一块发给代理即可。

**新来的？**

> 「帮我入门。」*（或附上任意格式的券商对账单——PDF、CSV、JSON、XLSX、截图、粘贴的文字——并说「带我 onboarding」）*

**想看看能做什么？**

> 「这里能做什么？」

**调整代理如何扮演你（风险偏好、仓位、禁区、语言、基准货币）：**

> 「带我过一遍设置。」
> 「看下我的 SETTINGS。」／「把我的基准货币改成 TWD。」

**记一笔交易或现金流：**

> 「昨天 185 美元买了 30 股 NVDA。」
> 「今天 400 美元卖出 10 股 TSLA。」
> 「GOOG 一季度股息 80 美元。」
> 「存入 5,000 美元。」
> 「这是我的 Schwab CSV——请导入。」

**问研究问题：**

> 「结合我当前组合分析 NVDA。」
> 「我现在 AI 敞口有多大？」
> 「财报前要不要减掉短线仓位？」

**生成今天的组合报告：**

> 「生成今天的组合体检。」
> 「跑我的盘前报告。」

代理在写入你的数据前会先确认，并按 `docs/` 下的合约端到端执行。你用日常语言描述即可；代理处理其余部分。

## 隐私

你的设置、交易数据库以及每一份生成的报告都留在本地——这些都不会被 git 跟踪。仓库里只有代理规格、示例模板和 Python 脚本。

## 第三方数据

价格流程可能使用公开市场数据与汇率接口（Stooq、Yahoo、Binance、CoinGecko、Frankfurter／ECB、Open ExchangeRate-API、TWSE／TPEx），以及你自行提供的可选 API 密钥。本项目不运营也不背书任何提供商——条款、速率限制与付费访问由你自行负责。
