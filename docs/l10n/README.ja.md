# 投資リサーチ・アシスタント（AI エージェント）

英語版 README がマスターです。その他の言語は読みやすさのための翻訳です。

このリポジトリは、お使いの PC 上で動く **AI 投資リサーチ・アシスタント**のワークスペースです。**Claude Code、OpenAI Codex、Gemini CLI**、またはファイルを読みコマンドを実行できるエージェント環境で開き、普段話す日本語で指示するだけで使えます。

**おすすめモデル：** **Claude Sonnet 4.6** を **High** の推論設定で使うか、同等以上の性能の新しいモデルを選んでください。軽めのモデルは手順を飛ばしたり、分析の深さが十分にならないことがあります。

## エージェントにそのまま頼む

コマンドやスキーマ、ファイル構成を覚える必要はありません。やりたいことに合わせて、下のどれかをコピーして貼り付けてください。

**はじめて使う？**

> 「始め方を教えて。」*（または証券会社の残高・取引明細などを、PDF／CSV／JSON／XLSX／画面キャプチャ／貼り付けテキストのどれでも添えて「オンボーディングして」と伝える）*

**ここで何ができるか知りたい？**

> 「ここで何ができる？」

**エージェントの振る舞いを自分に合わせる（リスク許容度、ポジションサイズ、触らない銘柄・領域、言語、基準通貨）：**

> 「設定を一緒に見て。」
> 「SETTINGS を確認して。」／「基準通貨を TWD に変えて。」

**売買や資金の出入りを記録する：**

> 「昨日 NVDA を 185 ドルで 30 株買った。」
> 「今日 TSLA を 400 ドルで 10 株売った。」
> 「GOOG の第 1 四半期配当、80 ドル。」
> 「5,000 ドルを入金した。」
> 「Schwab の CSV があるから取り込んで。」（他社・他フォーマットも `docs/` の契約に沿って扱います）

**リサーチの質問：**

> 「今のポートフォリオと突き合わせて NVDA を分析して。」
> 「今の AI 関連エクスポージャーはどれくらい？」
> 「決算を前に短期ポジションを減らすべき？」

**今日のポートフォリオレポートを作る：**

> 「今日のポートフォリオのヘルスチェックを出して。」
> 「寄り付き前に読む用のレポートを回して。」

データへの書き込みの前に必ず確認し、`docs/` 以下の契約ドキュメントに沿って最後まで処理します。普段の日本語で依頼すれば十分で、細かいところはエージェントが担います。

## プライバシー

設定、取引データベース（SQLite）、生成されたレポートはすべてローカルに残り、**Git には含まれません**。バージョン管理されるのはエージェント仕様、サンプルテンプレート、Python スクリプトのみです。

## サードパーティ・データ

価格取得では公開の市場データ・為替 API（Stooq、Yahoo、Binance、CoinGecko、Frankfurter／ECB、Open ExchangeRate-API、TWSE／TPEx など）および、あなたが設定した任意の API キーを使う場合があります。本プロジェクトはプロバイダーを運営も推奨もしません——利用規約、レート制限、有料プランは利用者ご自身の責任で確認してください。
