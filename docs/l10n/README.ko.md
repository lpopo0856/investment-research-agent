이 저장소는 AI 투자 리서치 에이전트용 로컬 작업 공간입니다. **Claude Code, OpenAI Codex, Gemini CLI** 또는 파일을 읽고 명령을 실행할 수 있는 에이전트에서 이 repo를 열고 대화하면 됩니다.

**모델 등급:** **Claude Sonnet 4.6**에 **High** 추론 강도를 쓰거나, 그에 버금가거나 더 능력 있는 최신 모델을 쓰세요. 더 가벼운 모델은 단계를 건너뛰거나 깊이가 약해질 수 있습니다.

## 에이전트에게 그대로 말하기

명령어, 스키마, 파일 구조를 미리 익힐 필요 없습니다. 원하는 일에 맞는 아래 문구를 골라 붙여 넣으면 됩니다.

**처음이신가요?**

> "시작하는 데 도와줘." *(또는 증권사 명세를 어떤 형식이든——PDF, CSV, JSON, XLSX, 스크린샷, 붙여넣은 텍스트——첨부하고 "온보딩 해줘"라고 하세요)*

**무엇을 할 수 있는지 보고 싶다면?**

> "여기서 뭘 할 수 있어?"

**에이전트가 나처럼 행동하도록 조정하기(위험 성향, 비중, 금지 영역, 언어, 기준 통화):**

> "설정을 같이 살펴보자."
> "내 SETTINGS 검토해 줘." / "기준 통화를 TWD로 바꿔 줘."

**거래나 현금 흐름 기록:**

> "어제 NVDA 30주를 $185에 샀어."
> "오늘 TSLA 10주를 $400에 팔았어."
> "GOOG 1분기 배당 $80."
> "$5,000 입금했어."
> "내 Schwab CSV야 — 가져와 줘."

**리서치 질문:**

> "지금 포트폴리오 기준으로 NVDA 분석해 줘."
> "지금 내 AI 익스포저가 얼마야?"
> "실적 발표 전에 단기 포지션 줄여야 할까?"

**오늘 포트폴리오 리포트 만들기:**

> "오늘 포트폴리오 헬스 체크 만들어 줘."
> "프리마켓 리포트 돌려 줘."

에이전트는 데이터에 쓰기 전에 확인하고, `docs/` 아래 계약 문서를 처음부터 끝까지 따릅니다. 평이한 말로 요청하면 되고, 나머지는 에이전트가 처리합니다.

## 프라이버시

설정, 거래 데이터베이스, 생성된 모든 리포트는 로컬에만 남으며 git에 포함되지 않습니다. 버전 관리되는 것은 에이전트 사양, 예시 템플릿, Python 스크립트뿐입니다.

## 서드파티 데이터

가격 워크플로는 공개 시세·환율 엔드포인트(Stooq, Yahoo, Binance, CoinGecko, Frankfurter / ECB, Open ExchangeRate-API, TWSE / TPEx)와 사용자가 제공하는 선택적 API 키를 쓸 수 있습니다. 이 프로젝트는 어떤 제공업체도 운영하거나 보증하지 않으며, 약관·속도 제한·유료 접근은 사용자 책임입니다.

## 면책

개인 리서치 전용이며 투자 자문이 아닙니다. 매매 전 중요한 사실은 반드시 직접 확인하세요.
